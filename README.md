# Web Programming Assignment 2

This is a responsive user registration web page built with plain HTML, CSS, and JavaScript. It features real-time input validation (for name, age, email, and password matching) and a clean, modern glassmorphic design.

## Folder Structure

```
Web_Programming_assignment_2/
├── index.html          # Main page containing the form and modal structures
└── assets/
    ├── css/
    │   └── style.css   # Custom CSS styling (variables, layouts, and responsiveness)
    └── js/
        └── script.js   # Client-side validation and modal control logic
```

## Features
* **Real-time Validation**: Validates full name, age (18–60), email format, and password fields as you type and on submission.
* **Success Modal**: Shows a custom interactive overlay confirming successful registration.
* **Responsive Design**: Designed with CSS flexbox/grid to display correctly on both mobile and desktop screens.
* **Pure Web Stack**: Built with zero external frameworks or dependencies.

## Running Locally
Simply double-click the `index.html` file to open it in any modern browser, or run it through a local development server (like VS Code Live Server or python's `http.server`).
